Fiids
=====

How to install/run
==================

Install Meteor

	curl https://install.meteor.com | /bin/sh

Create project

	meteor create fiidsapp

Copy source

	cd fiidsapp
	git clone https://github.com/xavo/fiids.git .

Add packages

        meteor add bootstrap
        meteor add jquery
        meteor add less
        meteor add accounts-base
        meteor add accounts-github
        meteor add accounts-ui

Run app

	meteor
